Patcher * Patching Programs

Patcheren er et universelt værktøj til at modificere filer eller diske. Det var det foretrukne værktøj for WHDLoad installationer til at lave filer eller disk images fra originale diske med sit eget disk format i tidligere versioner af WHDLoad pakken. Det foretrukne værktøj for denne opgave er nu RawDIC. For at lære mere om Patcheren burde du læse dets dokumentation, som også kan findes i Patcher arkivet leveret med WHDload's DEV pakke.

Patcher behøver ruby.8 fonten! Du kan finde denne font på originale Workbench diske. hvis fonten ikek er tilgængelig vil Patcheren simpelthen ikke starte.

Fordele:

  • nemt valg af kilde drev
  • diskroutine kan ikke fejle pga. for hurtig CPU
  • virker på escom-amiga'er med diskdrive-hardware 'feature'
  • multitasks mens den læser (det er endog muligt at installere 2 spil på samme tid hvis du har 2 disk drev)
  • kan vise det spor den loader (brugbart for bug reporter) og imaging problemer (læse fejl, disk fuld osv.
  • kan debugges med en normal debugger
Ulemper:
  • Brugte ~6-8 timer til at komme ind i det som programmør
  • måske større hukommelses forbrug end ~1 MB, afhængig af disk imaget (men for folk med mindre end 2 MB hdinstallationer er det heller ikke så brugbart)
