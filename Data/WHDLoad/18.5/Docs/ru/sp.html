SP * Сохранение картинок

Утилита предназначена для извлечения картинок из файла дампа, созданного WHDload.

Установка

SP содержится только в архиве DEV, но не в архиве USR.
Скопируйте в C: или другое подобное место

Использование

Сначала должен быть создан файл дампа. Запустите установленную программу, из которой предполагается вытащить картинку. Программа должна выполниться с параметрами Snoop/S, Expert/S и DebugKey/K/N. Если картинка, которую Вы хотите сохранить появилась, нажмите DebugKey. WHDLoad сохранит файл дампа. Используя Shell/CLI, войдите в каталог, где расположен файл дампа. Теперь запустите SP, введя в качестве параметра имя файла под которым будет сохранена картинка. SP загрузит файл дампа и попытается сохранить картинку. Если вы хотите установить параметр CoreDumpPath, то вам не нужно вводить директорию для дампа файлов, потому как для нахождения пути SP считывает файл глобальной конфигурации WHDLoad.

Параметры

OutputFile/A, Cop/K, CS=CopStop/K, Width/K, Height/K, con0/K, mod1/K, mod2/K, pt1/K, pt2/K, pt3/K, pt4/K, NoCopLst/S
OutputFile/A Имя файла для сохраняемой картинки
Cop/K Задаёт адрес начала интерпретирования первого копперлиста, если вы хотите перезаписать адрес содержащийся в файле дампа
CS=CopStop/K Задаёт адрес конца интерпретации копперлиста.
Параметр должен использоваться, если экран создаётся из более чем одного изображения.
Width/K Задаёт ширину сохраняемого изображения.
Не влияет на значения "modulo"!
Height/K Задаёт высоту сохраняемого изображения.
con0/K Перезаписывает регистр bplcon0, содержавшийся в файле дампа.
mod1/K, mod2/K Перезаписывает регистры bplmod1/bplmod2, содержавшиеся в файле дампа.
pt1/K,pt2/K, pt3/K,pt4/K   Перезаписывает регистры bplpt1,2,3,4, содержавшиеся в файле дампа.
NoCopLst/S Отключает дамп копперлиста.
