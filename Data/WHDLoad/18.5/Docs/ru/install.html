Установка WHDLoad

Самый простой способ установить WHDLoad и прилагаемые утилиты - это использовать инсталляционный скрипт, поставляемый с архивом.

Самое главное условие это, чтобы WHDLoad и утилиты были доступны из любого места без определения пути к ним. Для этого рекомендуется поместить их в "C: ". Также, WHDLoad может быть установлен в тот же каталог, где расположена запускаемая им программа (заметьте, что в демонстрационных файлах в иконках проектов используется только имя "WHDLoad" без пути).

Утилиты "DIC", "RawDIC" и "Patcher" также должны быть доступны, т.к. они требуются во время установки многих патчей. Хотя после того, как патчи уже установлены, данные утилиты не требуются для запуска самих приложений.
